# P4-Custom-Network-Telemetry

A programmable data-plane telemetry project using **P4** to implement custom network monitoring capabilities.

This repository explores the use of programmable switches and software-defined networking concepts to collect, process, and analyze network information directly in the packet processing pipeline.

P4 enables developers to define how packets are parsed, processed, modified, and forwarded inside programmable network devices. This makes it possible to implement custom telemetry mechanisms without relying only on traditional monitoring protocols. :contentReference[oaicite:1]{index=1}
